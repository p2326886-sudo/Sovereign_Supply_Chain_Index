# Sovereign Supply Chain Index
## Overview
Sovereign Supply Chain Index is a frontend intelligence dashboard designed to
analyze strategic supply chain resilience and sovereign infrastructure
dependencies.
The system focuses on:
- supply chain intelligence
- resilience indicators
- infrastructure analysis
- strategic dependency visualization
- economic system mapping
## Features
- Interactive intelligence dashboard
- Responsive frontend architecture
- Multi-section analytical interface
- TailwindCSS responsive design
- Vercel deployment integration
## Tech Stack
- React
- Vite
- TailwindCSS
- GitHub
- Vercel
